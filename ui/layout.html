CryptDown Info

CryptDown is a Microservice to store documents, pastes or code snippets, similar to pastebin.

CryptDown uses real 256bit AES encryption, which takes place directly in your browser. No plaintext is ever being send. Nobody can help you when you loose an encryption password for a paste. Your IP-Address will not be saved!

CryptDown was build with the following great and free software:

Fat-Free Framework (backend) CryptoJS MediumEditor Showdown.js to-markdown.js jQuery Twitter Bootstrap font-awesome

The encrypted document symbol was made by Ilsur Aptukov.
